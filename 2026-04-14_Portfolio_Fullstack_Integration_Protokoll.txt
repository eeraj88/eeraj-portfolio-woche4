                        TEIL 2: MAVEN INSTALLATION
═══════════════════════════════════════════════════════════════════════════════

Schritt 2.1: Maven herunterladen
─────────────────────────────────
1) Browser öffnen
2) Gehe zu: https://maven.apache.org/download.cgi
3) Lade herunter: "apache-maven-3.9.14-bin.zip" (Binary zip archive)
   NICHT: tar.gz (für Linux), src.zip (Quellcode), *.sha512, *.asc

Schritt 2.2: Maven entpacken
───────────────────────────
1) Rechtsklick auf apache-maven-3.9.14-bin.zip
2) "Alle extrahieren" oder "Extract All"
3) Es wird im Downloads Ordner entpackt: apache-maven-3.9.14

ERGEBNIS:
    Downloads/apache-maven-3.9.14/

Schritt 2.3: Maven verschieben
─────────────────────────────
1) Im Windows Explorer zu: C:\Program Files\
2) Neuen Ordner erstellen: "Maven"
3) Zum Downloads Ordner navigieren
4) Ordner "apache-maven-3.9.14" rechtsklicken → Ausschneiden
5. Nach C:\Program Files\Maven\ navigieren → Einfügen

ERGEBNIS:
    C:\Program Files\Maven\apache-maven-3.9.14\

Schritt 2.4: Umgebungsvariablen setzen
───────────────────────────────────────
1) Windows-Taste + R drücken
2) Eingabe: sysdm.cpl
3) Enter drücken
4) Reiter "Erweitert" klicken
5) Button "Umgebungsvariablen" klicken

6) Bei "Systemvariablen" (unten):
   ────────────────────────────────────────────────────────────────────────
   A) MAVEN_HOME erstellen:
      - Button "Neu" klicken
      - Variablenname: MAVEN_HOME
      - Variablenwert: C:\Program Files\Maven\apache-maven-3.9.14
      - OK klicken

   B) PATH bearbeiten:
      - "Path" auswählen
      - Button "Bearbeiten" klicken
      - Button "Neu" klicken (unten im Fenster)
      - Eingeben: %MAVEN_HOME%\bin
      - ENTER drücken
      - Bei allen Dialogen "OK" klicken (um alle Fenster zu schließen)

Schritt 2.5: Terminal neu öffnen
────────────────────────────────
WICHTIG: Alle Terminals schließen, komplett neues Terminal öffnen!

Schritt 2.6: Maven-Version prüfen
─────────────────────────────────
Im neuen Terminal eingeben:

    mvn --version

ERWARTETE AUSGABE:
    Apache Maven 3.9.14 (5xxxxxxx)

                        API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════════

Visitor API:
  GET  /api/visitor/count       → {"count": 123}

Like API:
  GET  /api/likes/{projectId}  → {"count": 5}

Comment API:
  GET  /api/comments/{projectId} → [{id, projectId, name, message, timestamp}, ...]
